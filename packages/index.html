how to

create an issue at packagedingus/packages with this template:

Name: packagename
Version: "1.0.0"
url: "package-url.package/package.tar.gz"

faq

  • how do i make a tar.gz?
    • 7zip, winrar, any zip creator pretty much that supports tar.gz
  • why did my [horrible] package get declined?
    • put effort into a package. if you don't, it won't get accepted, no exceptions.
  • how do i update my package?
    • make another issue, with a new version, but name and url the same, and add the "update" label.
  • the bot is broken!
    • submit an issue at packagedingus/error, and make sure it's actually a bug first.
